#### GIT Add

* `git add .`
* `git add --all`
* `git add FILENAME`
* `git add LIST OF FILES`
* `git add *.txt` - Add all .txt files in the current directory
* `git add docs/*.txt` - Add all .txt files in the docs directory
* `git add docs/` - Add all files in the directory and it's sub directory
* `git add "*.txt"` - Add all .txt files in the whole project
