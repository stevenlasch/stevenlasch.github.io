Git: Stash

  • To remove a listing: git stash drop stash@{0}
  • To view the stash stack: git stash list
  • To apply the most recent: git stash apply
  • To apply a stashed listing: git stash apply stash@{0}
  • To Show a stashed listing: git stash show -p stash@{0}
  • Create a new stash: git stash create "message"
