  • If I'm using Python HTTP server, how do I modify the end_headers to prevent file caching?

Simple Web Server

Windows

  1. Navigating to the directory containing your HTML files
  2. In a command terminal, run: `python -m http.server 9000`
  3. Go to http://localhost:9000

Mac: (Python 2.7.10)

  1. Open terminal: cd /Documents/workspace/data
  2. In a command terminal, run: `python -m SimpleHTTPServer 9000`
  3. Go to http://localhost:9000
